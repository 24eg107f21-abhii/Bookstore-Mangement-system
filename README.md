# 📚 Readers Home Page

A simple HTML homepage for the **Readers Store** project developed as part of a Web Programming Lab.

## 📖 Description

This project creates a basic homepage for the **Anurag University Readers Store**. It includes navigation links to the Home, Login, and Contact Us pages.

## 🚀 Features

- Simple HTML5 webpage
- Centered layout
- Navigation menu
- Easy to understand for beginners

## 🛠️ Technologies Used

- HTML5

## 📂 Project Structure

```
Readers-Home-Page/
│── index.html
│── login.html
└── README.md
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.

## 📸 Output

The homepage displays:

- **WELCOME TO READERS**
- **Anurag University Readers Store**
- Navigation links:
  - HOME
  - LOGIN
  - CONTACT US

## 📚 Learning Objectives

- Understand HTML document structure
- Use headings and hyperlinks
- Create a basic webpage
- Learn simple website navigation
